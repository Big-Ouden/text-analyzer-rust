## 💾 Installation
1. Clone the repository:
```
git clone https://github.com/Big-Ouden/text-analyzer-rust.git
cd text-analyzer-rust
```
2. Build the project:
```
cargo build --release
```
3. Run the binary:
```
./target/release/text-analyzer-cli --help
```
